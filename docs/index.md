## Download

- [Chrome Web Store](https://chrome.google.com/webstore/detail/slack-emoji-meister/omcnknklnilbbnoioiaibdkhoonlmdnj)

## How to use

1. Right click any image
2. Select 'add slack image'
3. Enter your slack team name
4. Enter emoji name
